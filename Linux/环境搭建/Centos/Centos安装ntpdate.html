  • 阿里云 NTP
  • 阿里云 NTP

在Linux系统中，您可以通过ntpdate和ntpd两种命令方式实现NTP时间同步。

  • ntpdate为断点更新。对新购实例，您可以使用ntpdate同步时间。

  • ntpd为步进式地逐渐调整时间。对已经承载有运行中业务的实例，建议您使用ntpd同步时间。

shell
yum install ntpdate -y

shell
